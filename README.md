# Registration Form 📝

A simple and responsive registration form built with **HTML** and **CSS**, created as part of the FreeCodeCamp tutorial.

## 📋 Features

- Collects user's:
  - First name
  - Last name
  - Email
  - Password
  - Account type
  - Age
  - Bio
  - Profile picture
- Styled with a dark theme
- Fully responsive layout

## 🚀 Technologies Used

- HTML5
- CSS3 (custom styling)
